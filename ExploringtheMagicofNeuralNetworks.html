Problem Statement: Consider you have this dataset


Dataset

For instance, if you enter a value of -10, your output will be 14, if you enter a value of 0, your output will be 32, and so on. I want to ask you a question while you look carefully at these numbers.


Predict value 30

Can you predict what the output value will be if you use 30 as the input value? Before we show the solution, please take some time to find it.


Found the answer

You are correct, if you found the answer 86. Now I explain how this problem might be solved using both traditional programming and machine learning. Well, if you carefully look at the input and outputs, you will see that they are connected by the temperature conversion equation.


Celsius to Farenheit

F = C x 1.8 + 32

F = Fahrenheit C = Celsius


F equals C times 1.8 plus 32. Where the outputs refer to degree in Fahrenheit and are represented here by the letter F, and the inputs refer to degree in Celsius and are represented here by a letter C. In other words, the inputs represent a temperature value in degree Celsius, and the outputs display the corresponding temperature in degree Fahrenheit.


For example, an input value of zero refers to a temperature of zero degree Celsius, which corresponds to 32 degree Fahrenheit.


F = 0 x 1.8 + 32


we get 32 by multiplying zero with 1.8 and adding 32 like we see here in this equation. Similarly, an input value of 10 refers to a temperature of 10 degree Celsius which corresponds to 50 degree Fahrenheit. we get 50 multiplying 10 with 1.8 and adding 32 using the above equation.

What your brain did to find out the relationship between these specific inputs and outputs is exactly what machine learning algorithms performs. In other words, given a set of inputs and the corresponding outputs, machine learning algorithms will find the hidden patterns and relationship between the inputs to the outputs.


Traditional Programming vs Machine Learning

Traditional Approach


In traditional software development, the input and algorithm are known, and you write a function to produce an output. On the other hand, with machine learning, both the input and the output are known, but the algorithm that generates the output given the input is unknown, So the algorithm in machine learning needs to be learned.
